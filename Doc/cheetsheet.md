# Cheat Sheet

## buffer

- <leader> + B        : 新規バッファを開く
- <leader> + q        : カレントのバッファを閉じる
- <leader> + n        : 次のバッファに移動
- <leader> + p        : 前のバッファに移動
- <leader> + <leader> : 前に選択していたバッファを循環移動

## winresizer plugin

- 'Ctrl + e'  : 画面リサイズ開始
  - f         : 画面選択モード
  - m         : 移動モード開始
  - r         : リサイズモード開始
